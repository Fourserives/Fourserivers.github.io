使用最小花费爬楼梯

题目：https://leetcode.cn/problems/min-cost-climbing-stairs/

解题思路：

1、dp数组以及下标含义

dp[i]：第i阶楼梯所需要的最小花费dp[i]

2、递推公式

cost[i]是i阶楼梯跳跃所需要的花费，i则是表示第i阶的楼梯

则当从第1阶楼梯进行跳跃，可以通过支付cost[1]来实现

当从第2阶楼梯进行跳跃，可以通过支付cost[2]来实现

如果要跳跃到第三阶，则有以下可能：

从第一阶楼梯支付，跳跃两层：dp[3] = dp[1] + cost[1]

从第二阶楼梯支付，跳跃一层：dp[3] = dp[2] + cost[2]

其中dp[1]表示到达第一阶已经花费的数量，dp[2]表示到达第二阶已经花费的数量

综上所述，可以得到递推公式为：

dp[i] = min( (dp[i-1]+cost[i-1])，(dp[i-2]+cost[i-2]))

3、dp数组初始化

由于题目需求是可以选择从第0阶或者第1阶开始爬，所以：

dp[0] = 0，dp[1] = 0

4、遍历顺序

从前往后

买卖股票的最佳时机

题目：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/

解题思路：

1、dp数组以及下标含义

dp[i ] [0]：持有股最大值

dp[i ] [1]：无持股最大值

2、递推公式

由于dp[i ] [0]表示持有股所得现金最大值，所以dp[i ] [0]最大值是在两种情况下取：

第一种情况：dp[i-1] [0]也就是之前就买入股的现金

第二种情况：- prices[i]也就是刚买入股的现金

由于dp[i ] [1]表示无持股所得现金最大值，所以dp[i ] [1]最大值是在两种情况下取：

第一种情况：dp[i-1] [1]也就是之前已经把股票售出(所以是无持股)的现金

第二种情况：dp[i ] [0]+prices[i]也就是刚售出股的现金

综上所述，递推公式即：

dp[i] [1] = max(dp[i-1] [1]，dp[i ] [0]+prices[i])

其中，dp[i ] [0] = max(dp[i-1] [0]，-prices[i])

3、初始化

由递推公式可知，初始要从dp[0 ] [0]，dp[0 ] [1]开始，所以初始化为：

dp[0 ] [0] = -prices[0] ， 由于dp[0 ] [0]表示持有股且它为首位，所以只能取第一天就买入股消费的现金，因为只有这样才满足我们对dp[i] [0]的定义！

dp[0 ] [1] = 0，由于dp[0 ] [1]表示无持股且它为首位，所以取第一天没买入股，也就是0

买卖股票的最佳时机III

题目：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iii/description/

解题思路：

1、dp数组以及下标含义

由于该题为至多两次买卖股票，所以dp数组用二维来方便记录如下情况：

dp[i] [0] ：不操作时现金

dp[i] [1]：第一次持有股票的最大现金

dp[i] [2]：第一次无持股的最大现金

dp[i] [3]：第二次持有股票的最大现金

dp[i] [4]：第二次无持股的最大现金

2、递推公式

dp[i] [0]：若不进行任何操作，则dp[i] [0] = 0

dp[i] [1]为第一次持有股票的最大现金，所以有两种可能：

第一种为之前就已经第一次持有股票dp[i-1] [1]，

第二种为这次刚买入第一次股票dp[i-1] [0] - prices[i]

则dp[i ] [1] = max(dp[i-1] [1]，dp[i-1] [0] - prices[i])

dp[i] [2]为第一次无持股的最大现金，所以有两种可能：

第一种为之前就已经把第一次股票出售出去，dp[i-1] [2]，

第二种为这次刚把第一次买入的股票出售出去，dp[i-1] [1] + prices[i]

dp[i] [2] = max(dp[i-1] [2]，dp[i-1] [1] + prices[i])

dp[i] [3]为第二次持有股票的最大现金，所以有两种可能：

第一种为之前就已经第二次持有股票dp[i-1] [3]，

第二种为刚买入第二次股票dp[i-1] [2] - prices[i]

dp[i] [3] = max(dp[i-1] [3]，dp[i-1] [2] - prices[i])

dp[i] [4]为第二次无持股的最大现金，所以有两种可能：

第一种为之前就已经第二次无持股dp[i-1] [4]，

第二种为刚把第二次股票售出去，dp[i-1] [3] + prices[i]

dp[i] [4] = max(dp[i-1] [4]，dp[i-1] [3] + prices[i])

3、初始化

dp[0] [0]：无操作，dp[0] [0] = 0

dp[0] [1]为第一次持有股，即dp[0] [1] = -prices[0]

dp[0] [2]为第一次不持有股，即dp[0] [1] = 0

dp[0] [3]为第二次持有股，即dp[0] [2] = -prices[0]

dp[0] [4]为第二次不持有股，即dp[0] [4] = 0

打家劫舍

题目：https://leetcode.cn/problems/house-robber/

解题思路：

1、dp数组以及下标的含义

dp[i]：小偷在第i个房间内能够取得的最高金额dp[i]

2、递推公式

由题目可以推测，假设以第三间房间来算小偷的最高金额，则有以下两种情况：

第一种，小偷进入第一间房间偷取金额dp[0]，间隔一间后偷取本间房间的金额nums[2]

第二种，小偷进入第二间房间偷取金额dp[1]，第三间为间隔所以无法偷取

所以第三间的金额计算就是 max(dp[0]，(dp[1]+nums[2]))

再以第四间房间来算，则两种情况分别是第三间的最高金额dp[2]与第二间的最高金额dp[1]加上相隔一间后本间金额nums[3]

综上所述，可以得到递推公式为：

dp[i] = max(dp[i-1]，(dp[i-2]+nums[i]))

3、初始化

由于dp数组的意义，所以第一间房间金额必然是dp[0] = nums[0]

而第二间房间比较特殊，由于递推公式需要有i-2，如果直接代入公式会导致数组下标为负数，不合法，结合题目情况，有：

dp[1] = max(dp[0],nums[1])
